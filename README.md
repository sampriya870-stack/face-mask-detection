# Face Mask Detection using Deep Learning

## Project Objective
This project aims to build a deep learning model that can detect whether a person is wearing a face mask or not using image classification techniques.

## Tools and Technologies
- Python
- TensorFlow / Keras
- OpenCV
- Deep Learning
- Computer Vision

## Dataset
Image dataset containing masked and unmasked face images.

## Project Steps
1. Data preprocessing
2. Image classification model development
3. Training deep learning model
4. Testing and prediction

## Results
The model can identify whether a person is wearing a mask or not from an image.

## Applications
- Public safety monitoring
- Smart surveillance systems
- Health safety enforcement
